.i3-config
==========

My configs and shell scripts collection for i3wm.

Dependencies:
=============

- gnome-terminal (you can change it to your favorite terminal)
- dmenu
- conky
- i3lock
- amixer
- setxkbmap
- xbelld
- numlockx
- udiskie
- feh
- jdate 
- i3-wm :D

Installation:
=============

``$ cd ~ # sure you can run "cd" itself :D``

``$ git clone git://github.com/omid/.i3-config.git .i3``
